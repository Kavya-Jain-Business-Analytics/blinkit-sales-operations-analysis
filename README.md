# Blinkit Sales & Operations Analysis

## Project Overview

This project is a business analytics analysis of Blinkit sales and outlet performance using Microsoft Excel and Microsoft Power BI.

The project involved working with raw sales data, preparing the dataset for analysis, analysing key performance indicators, developing PivotTable-based analysis, and creating interactive dashboards to evaluate sales and outlet performance.

---

## Project Objective

The objective of this project was to analyse sales performance and understand the factors contributing to outlet-level and product-level performance.

The analysis focused on evaluating revenue, sales volume, outlet characteristics, product categories, and other key business performance indicators.
